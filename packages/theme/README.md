# VF-UI theme

VF-UI component theme

## Installation

```shell
npm install --save vfui-theme
```

## Usage

Use Less import

```less
@import "vfui-theme";
```

Or Use webpack

```js
import "vfui-theme";
```

Or Use In UMD

```html
<link rel="stylesheet" path="path/to/node_modules/vfui-theme/lib/index.css" />
```

## Import on demand

```js
import "vfui-theme/lib/input.css";
```
