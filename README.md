## Project setup

```bash
npm install

// run example
npm run dev
```

## How to use

```
npm install --save vf-vue-ui
```

```js
import Vue from "vue";
import VFUI from "vf-vue-ui";
import "vf-vue-ui/lib/theme/index.css";

// 安装组件
Vue.use(VFUI);
```
